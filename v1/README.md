# Terrain Portfolio — Version 1 (frozen)

This folder is a **preserved snapshot** of the first complete version of the site.

## Run v1

```bash
cd v1
npm install
npm run dev
```

Open **http://localhost:3000** (or the port shown in the terminal).

## Note

Active development continues in **`../v2`**. Do not edit this folder unless you intentionally want to change the v1 archive.
