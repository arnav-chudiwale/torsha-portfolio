# Terrain QR codes

Scannable link: **https://torsha-portfolio.vercel.app/**

| File | Use |
|------|-----|
| `terrain-qr-card-light.png` | Print / presentations (field-paper, day mode) |
| `terrain-qr-card-dark.png` | Dark backgrounds, slides |
| `terrain-qr-share.png` | Social posts (1080×1080) |
| `terrain-qr-standalone.png` | QR only (embed in documents) |

Regenerate after URL changes:

```bash
python scripts/generate-terrain-qr.py
```
